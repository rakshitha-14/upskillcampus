# 🛒 Grocery Delivery Application

A full-stack web application that allows users to browse grocery items, place orders, and manage deliveries efficiently. This project demonstrates the implementation of a modern e-commerce platform using frontend and backend technologies.

---

## 📌 Features

- 🛍️ Browse grocery products
- ➕ Add items to cart
- 🧾 Place orders
- 📦 View order details
- 🔐 User-friendly interface
- ⚡ Fast and responsive design
